# AI-GPA-Calculator

一个基于 AI 的绩点计算器 —— 上传成绩单图片或 PDF，AI 自动提取课程信息，选择绩点标准后一键计算 GPA。

## 功能

1. **AI 表格提取**：上传成绩单图片（JPG/PNG）或 PDF，通过 OpenRouter API（Gemini-2.5-Flash）自动识别并提取课程名称、学分、成绩
2. **人工校验**：提取结果以可编辑表格展示，支持修改和新增行，不确定的字段会提示用户重点检查
3. **多种绩点标准**：内置标准 4.0、改进 4.0(1)、改进 4.0(2)、北大 4.0、加拿大 4.3、标准 5.0 共 6 种标准，还支持用户自定义
4. **多标准对比**：一键用所有标准计算 GPA 并列对比，帮你选择最有利的标准
5. **导出 CSV**：计算结果可导出为 CSV 文件

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 配置 API Key（二选一）
# 方式一：复制 .env.example 为 .env 并填入你的 Key
cp .env.example .env
# 方式二：在应用侧边栏直接输入

# 启动
streamlit run app.py
```

API Key 从 [OpenRouter](https://openrouter.ai/keys) 获取。

## 绩点标准

见 `standard.png`，支持以下标准：

| 标准 | 满分绩点 | 说明 |
|------|---------|------|
| 标准 4.0 | 4.0 | 经典 90/80/70/60 四档 |
| 改进 4.0(1) | 4.0 | 85/70/60 三档 |
| 改进 4.0(2) | 4.0 | 85/75/60 三档 |
| 北大 4.0 | 4.0 | 细分 10 档 |
| 加拿大 4.3 | 4.3 | 8 档制 |
| 标准 5.0 | 5.0 | 公式制 (分数-50)/10 |

## 隐私说明

- 上传的成绩单数据不会存储在任何服务器上
- 数据仅通过 OpenRouter API 临时处理后即丢弃
